Tech Stack: 핵심 기술 설명

Gemini Grounding

LLM에 실시간 Google 검색 능력 부여. 학습 데이터 이후 정보도 검색 가능.

TanStack Query

React 서버 데이터 관리. 캐싱, 자동 재시도, 백그라운드 갱신 자동화.

Celery + Redis

무거운 작업을 백그라운드 Worker에서 처리. 웹서버 블로킹 없음.

ThreadPoolExecutor

Python 병렬 처리. 3개 Agent와 2개 검색을 동시 실행.

Circuit Breaker

Provider별 장애 자동 격리. 3회 실패 시 5분간 차단 후 Fallback.

pgvector

PostgreSQL 벡터 검색. 유사 케이스 검색으로 인사이트 품질 향상.
